Đáng mua hay nên chờ?

Dán link sản phẩm Shopee để phân tích giá sau voucher, tiết kiệm thực và Deal Score — lấy trực tiếp từ dữ liệu Shopee, không đoán mò.

1) Dán link ở trên rồi bấm "Phân tích trên GitHub" — link sẽ tự copy vào clipboard, tab GitHub Actions mở ra.
2) Trong tab đó bấm nút xanh Run workflow → dán link vào ô product_url → bấm Run workflow lần nữa để chạy thật.
3) Đợi khoảng 1 phút cho chạy xong, quay lại đây bấm "Tải kết quả".

⚙️ Hoặc chạy bằng terminal (dành cho dev, có sẵn cookie local)

Copy lệnh, dán vào terminal dự án (cần node + SHOPEE_COOKIES trong .env), Enter, rồi quay lại bấm "Tải kết quả".

📊

Chưa có kết quả phân tích nào. Dán link sản phẩm ở trên rồi bấm "Phân tích trên GitHub".

🧪 Dữ liệu giá/voucher lấy trực tiếp từ API sản phẩm của Shopee lúc phân tích. Mục "Ước tính" (Flash Sale, xu hướng giá) là suy luận, không phải số liệu chính thức.
